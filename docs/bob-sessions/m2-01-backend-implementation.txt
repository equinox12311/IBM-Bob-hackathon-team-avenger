## Task Description
Implement core backend API with FastAPI, SQLite + sqlite-vec storage, and semantic search capabilities using sentence-transformers embeddings.

## Bob's Contributions

### 1. FastAPI Server (cortex_api/server.py)
Bob generated a production-ready FastAPI application with:
- RESTful API endpoints for entries, search, timeline
- Server-Sent Events (SSE) for real-time updates
- CORS middleware configuration
- Health check endpoint
- Comprehensive error handling
- Request/response logging

### 2. Storage Layer (cortex_api/storage.py)
Bob implemented SQLite database with vector search:
- Schema design for entries, feedback, wellness, automations
- sqlite-vec integration for 384-dimensional embeddings
- Vector similarity search with L2 distance
- Fallback to Python cosine similarity if extension unavailable
- Transaction management
- Connection pooling

### 3. Retrieval System (cortex_api/retrieval.py)
Bob created semantic search with re-ranking:
- Over-fetch strategy (k×3 candidates)
- Re-ranking formula: `score × exp(-λ×days) × cosine_similarity`
- Recency decay (λ = 0.005/day)
- Score clamping [0.1, 5.0]
- Feedback integration (boost +0.2, flag -0.3)

### 4. Embeddings Provider (cortex_api/embeddings.py)
Bob implemented flexible embedding system:
- Primary: sentence-transformers (all-MiniLM-L6-v2)
- Optional: watsonx.ai Granite embeddings
- Provider abstraction for easy switching
- Batch processing support
- Error handling and fallbacks

### 5. MCP Tools (cortex_api/tools.py)
Bob generated 5 MCP tools for IBM Bob integration:
- `diary_save`: Save journal entries with secret detection
- `diary_recall`: Semantic search over entries
- `diary_timeline`: Recent entries (reverse-chronological)
- `diary_link_code`: Attach code locations to entries
- `diary_feedback`: Boost/flag entries for learning loop

### 6. Data Models (cortex_api/models.py)
Bob created Pydantic models for:
- Entry creation and response
- Search requests and results
- Profile management
- Analytics data
- Automation rules
- Type safety throughout

### 7. Configuration (cortex_api/config.py)
Bob implemented environment-based settings:
- Database path configuration
- Embeddings provider selection
- LLM provider configuration
- API authentication
- Feature flags
